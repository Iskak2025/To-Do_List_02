# To-do List App

Простое веб-приложение **To-do List** на **HTML**, **CSS** и **JavaScript** с поддержкой **LocalStorage** и **интеграции с API**.  

Позволяет легко управлять задачами: добавлять, отмечать как выполненные, удалять и фильтровать. Данные синхронизируются с mock API и сохраняются в браузере.

Figma: https://www.figma.com/design/zOVEeMuHIfeu2RTFU9yW0P/Untitled?node-id=0-1&t=q32840NjMRq9Y4nY-1
---

## Возможности

- **Добавление задач**: Быстро добавляйте новые задачи.
- **Отметка выполненных задач**: Переключайте статус задачи между выполнено/не выполнено.
- **Удаление задач**: Удаление отдельной задачи, выбранных или всех задач сразу.
- **Фильтрация задач**: Просмотр всех задач, только ожидающих или выполненных.
- **LocalStorage**: Задачи сохраняются в браузере и не теряются при перезагрузке.
- **API интеграция**: Загрузка задач с mock API и синхронизация изменений (добавление/редактирование/удаление).
- **Адаптивный дизайн**: Приложение корректно работает на мобильных и десктопах.

Использование

Введите задачу в поле ввода и нажмите Add или клавишу Enter.

Нажмите на кружок, чтобы отметить задачу как выполненную.

Нажмите на значок корзины, чтобы удалить задачу.

Используйте кнопку Delete Selected для удаления выполненных задач.

Используйте кнопку Delete All для удаления всех задач.

Используйте выпадающий список Filter для фильтрации задач.

Нажмите Load API tasks, чтобы загрузить задачи с mock API.
